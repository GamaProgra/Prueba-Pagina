Hola

Soy Gamaliel. Se bienvenido/bienvenida a mi sitio web, donde encontraras información relacionada con mis habilidades y conocimientos en programación, desde la estrcutura basica de una página web en HTML, hasta distintos tipos de diseño en CSS.

HTML

Es un lenguaje de programación que permite la creación y formación de la estructura de una página web. Aquí podemos colocar el contenido deseado, porque podemos asignarles a todos los elementos un lugar dentro de la página. Puedo crear una página desde cero y/o trabajar con algo ya elaborado, colocando cada uno de los elementos solicitados, ya que tengo la capacidad de manejar una gran variedad de etiquetas y comandos dentro de HTML, ya que todas las herramientas que proporciona son muy útiles para la organización de cada uno de los elementos que tendrá la página. Trabajo con textos, imágenes, videos y audios e incluso algunos decorados.

CSS

Ofrezco un buen trabajo con CSS, ya que ayuda a expandir cualquier herramienta visual con HTML, puedo manipular una gran variedad de herramientas que me permiten trabajar con muchos elementos, así como en HTML, desde textos hasta videos. Una diferencia notable de HTML, es que el manejo de los elementos es mas extenso, así que puedo asignar un color, tamaño, orientación, tipo de letra, o cualquier tipo de añadidura para la página, dando a lugar una posibilidad mas grande para que el entorno tenga todo un apartado grafico excelente y la visualización sea agradable para el usuario.
